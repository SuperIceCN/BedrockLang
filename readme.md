# 什么是BedrockLang  

BedrockLang (简称BDL) 是为《我的世界：基岩版》专门设计的创意开发编程语言。  
BDL是简单、专一、强类型、高性能的，采取与mc命令类似的语法，对大部分mc原版命令直接兼容。
